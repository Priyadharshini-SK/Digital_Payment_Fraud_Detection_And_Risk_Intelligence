# Dashboard Screenshots

This folder contains screenshots of all three Power BI dashboards created for the project 
**Digital Payment Fraud Detection & Risk Intelligence**.

## Dashboard 1 – Executive Fraud Overview
This dashboard provides a high-level summary of digital payment fraud, including key KPIs 
such as total transactions, fraud transactions, fraud percentage, and total amount.

## Dashboard 2 – Fraud Pattern & Risk Analysis
This dashboard focuses on identifying fraud patterns based on transaction type, time, 
amount range, and risk categories to support operational and risk teams.

## Dashboard 3 – Customer & Risk Intelligence
This dashboard analyzes fraud trends by customer age group, device type, and geography, 
helping in targeted fraud prevention strategies.
